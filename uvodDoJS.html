Charakteristika jazyka JavaScript

» Je objektovo orientovaný scriptovací jazyk,

» Vytvoril ho Brendan Eich v roku 1995,

» Pôvodný názov bol LiteScript,

JavaScript je tzv. interpretovaný jazyk, to znamená, že script je prekladaný za behu priamo v prehliadači. Beží na strane klienta, čiže všetky scripty sú vykonávané v samotnom prehliadači. Vďaka tomu môžeme pomocou jazyka JavaScript dynamicky meniť obsah webovej stránky. Ako napríklad dropdown (roletové) menu, vyskakovacie okná, interaktívna obrázková galéria alebo bežiaci čas na stránke. Dôležitou vlastnosťou JavaScriptu je multiplatformovosť, čo znamená, že script pobeží na akomkoľvek zariadení s internetovým prehliadačom - počítač, smartfón, tablet ale i čoraz populárnejšie smart hodinky.

Prvý script

Script je počítačový program, pozostávajúci z príkazových riadkov rovnako ako akýkoľkovek program, napísaný napríklad v jazyku C++. Rozdieľ je len v tom, že program v C++ treba najskôr kompilátorom preložiť a až potom je možné ho spustiť, kdežto script je prekladaný interpretom prehliadača a ihneď aj vykonávaný samotným prehliadačom.

Podobne ako pri vytváraní HTML alebo CSS súborov, môžeme pre zápis scriptov použiť akýkoľkovek textový editor - napr. NotePad++, Sublime, Visual Studio Code a pod. Odporúčam ale použiť editor, ktorý je súčasťou nejakého vývojového prostredia. Takýto editor totiž poskytuje programátorovi komfort, napr. v dopĺňaní názov tried, objektov, funkcií, premenných a pod. Veľmi vhodný editor je NetBeans alebo Visual Studio Code, ktoré sú zadarmo na stiahnutie.

Ako teda vytvoriť prvý script?

<body>
 <script type="text/javascript">
  document.writ("Ahoj svet!");
 </script>
<body>


» atribút type určuje, že ide o zdrojový kód jazyka JavaScript,

» script obsahuje jeden riadok kódu, je ukončený bodkočiarkou, v JavaScripte nie sú povinné, ale my ich budeme používať,

» document je názov objektu a určuje, že ide o HTML dokument, čiže našu stránku,

» write() je metóda objektu document a zabezpečuje výpis textu "Hello World".


Ak si teraz otvoríme našu stránku, tak prehliadač ju zobrazuje zhora dole a keď narazí na značku <script>, spustní nami vytvorený script.

Výpis údajov

V tejto podkategórii si ukážeme rôzne výpisy údajov, ako sú celé a desatinné čísla, znaky a reťazce. Cieľom je naučiť sa metódu write().
Všetky hodnoty, ktoré budeme vypisovať, budú zapísané priamo v zdrojovom kóde. Takýmto hodnotám hovoríme literály. Lieterál je nepomenovaná konštantná hodnota, zapísaná priamo v kóde.

//celé čísla
document.write(563);
document.write(-79);
document.write(45 + 215);

//desatinné čísla
document.write(45.678);
document.write(3.890);
document.write(12.145);

//znaky - píšeme v apostrofoch ''
document.write('W');
document.write('C');
document.write('R');

//reťazce - píšeme v úvodzovkách ""
document.write("Programujeme v JavaScripte");
