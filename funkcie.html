Funkcie

Pri algoritmizácii sa snažíme každý problém rozložiť na čiastkové podproblémy, ktoré keď vyriešime v správnom poradí, tak vlastne vyriešime celý problém. Na riešenie takýchto podproblémov používame podprogramy. Je to relatívne samostatná programová jednotka, ktorá rieši určitý čiastkový problém. Podprogramy využívame hlavne vtedy, keď potrebujeme nejakú činnosť vykonať viacnásobne na rôznych miestach programu.

V JavaScripte sú podprogramy realizované funkciami a metódami. Doteraz sme sa stretli s niektorými funkciami a metódami, ako sú Math.sqrt(), Math.pow(), atď.. Tieto metódy sú zabudované v JavaScripte, hovoríme im štandartné funkcie a metódy.

Programátor má možnosť definovať svoje vlastné funkcie a vlastné metódy. Funkciu musíme najskôr deklarovať a definovať jej správanie - vlastne, čo má vykonať. Pri deklarácii určíme meno funkcie, či bude alebo nebude mať parametre.

Deklarácia a definícia funkcie

Na rozdiel od štandartných funkcií, musíme vlastné funkcie definovať sami. Všeobecný tvar deklarácie a definície funkcie je následovný:

function menoFunkcie(parametre) {
 //TODO kód pôjde sem
 return hodnota;
}

→ function - rezervované slovo,

→ menoFunkcie - identifikátor, ktorý volí sám programátor,

→ parametre - väčšinou premenné, do ktorých sa uložia hodnoty pri volaní funkcie,

→ hodnota - hodnota, ktorú funkcia vráti.

//Príklad deklarácie a definície funkcie
function sucet(s1, s2) {
 var suc = s1 + s2;
 return suc;
}

//Príklad volania funkcie
var vysledok = sucet(10, 5); //Vypíše 15
