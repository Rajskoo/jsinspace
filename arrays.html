Arrays (Polia)

Čo sa v tejto kapitole naučíme?

→ Deklarovať a vytvárať polia,

→ Používať polia.


Pole

 Pole je štruktúrovaný údajový typ. Na rozdiel od obyčajnej premennej, obsahuje niekoľko hodnôt, pričom v JavaScripte jednotlivé hodnoty nemusia byť rovnakého typu.
 Pole si môžeme predstaviť ako skupinu viacerých premenných, ktorým hovoríme prvky. Každý prvok má svoje poradové číslo, nazývané tiež index. Prvý prvok v poli má index 0, druhý index 1 a pod.
 Podobne ako obyčajné premenné aj pole je potrebné deklarovať. V JavaScripte existuje niekoľko spôsobov ako deklarovať pole, my budeme používať nasledovnú všeobecnú deklaráciu:

var menoPola = new Array();

Príklad:

//vytvoríme nové pole "cisla", ktoré neobsahuje žiadne prvky
var cisla = new Array();

//následne pridáme prvky do poľa
cisla[0] = 0;
cisla[1] = 5;
cisla[2] = 10;
cisla[3] = 15;
cisla[4] = 20;
cisla[5] = 25;

Počet prvkov

Počet prvkov zisťujeme vlastnosťou length.

document.write(cisla.length);

Použitie poľa

Pri poliach používame cykly veľmi často. Je to najmä vtedy, keď potrebujeme prechádzať cez celé pole - zápis do poľa, výpis poľa, hľadanie hodnoty a pod.

//zápis do poľa pomocou cyklu
for (var i = 0; i < 4; i++) {
 cisla[i] = prompt("Zadajte číslo: ");
}

//výpis poľa
for (var i = 0; i < cisla.length; i++) {
 document.write(cisla[i] + " ");
}
document.write("<br>");

//nájdenie najväčšieho čísla v poli
var max = celeCisla[0];
for (var i = 1; i < celeCisla.lenght; i++) {
 if (celeCisla[0] > max) {
  max = celeCisla[i];
 }
}
document.write("Najväčšia hodnota je: " + max);

Pre prácu s poľami je k dispozícii množstvo metód, napr.:

 concat() → spájanie polí,

 push() → pridanie prvku na koniec poľa,

 unshift() → pridanie prvku na začiatok poľa,

 shift() → odstránenie prvého prvku poľa,

 pop() → odstránenie posledného prvku poľa,

 splice() → odstránenie konkrétneho prvku poľa,

 sort() → triedenie prvkov poľa.
